# elcapitan-xfwm

MacOS like theme for XFWM4

![Example image](https://i.imgur.com/2RqCf7K.png)

## Installing

    mkdir ~/.themes
    cd ~/.themes
    git clone https://github.com/micopiira/elcapitan-xfwm.git
